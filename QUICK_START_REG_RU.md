## 🚀 4 простых шага:

### 1. Загрузи файлы на хостинг

**Через FTP или File Manager:**
- Все файлы проекта → в папку `public_html` (или `www`)
- Сохрани структуру папок

### 2. Создай файл `.env` в корне сайта

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_CHAT_ID=ваш_chat_id
START_DATE=1 ноября
```

**⚠️ ВАЖНО:** `.env` уже в `.gitignore`, не коммить его!

### 3. Настрой SSL

В панели REG.RU:
- **SSL** → активируй бесплатный Let's Encrypt
- Включи HTTPS (опционально)

### 4. Проверь работу

- Открой: `https://mustang-29.ru`
- Проверь форму отправки
- Проверь загрузку даты
